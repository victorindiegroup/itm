ITM — Convertisseur CSV Yokitup

Cette page convertit ton export CSV Yokitup en catalogue.json à uploader sur GitHub. Tout se passe localement dans ton navigateur, rien n'est envoyé en ligne.

3. Télécharge le catalogue

Le fichier catalogue.json est prêt. Clique pour le télécharger, puis uploade-le sur GitHub dans ton repo ITM (il remplacera l'ancien).

Étape suivante : sur GitHub, vas dans ton repo ITM → clique sur le fichier catalogue.json → icône crayon ("Edit this file") → supprime tout le contenu → colle le nouveau contenu OU plus simple : upload nouveau fichier qui écrase l'ancien. Commit, et c'est déployé.
Mémo overrides appliqués

    📖 Règles appliquées

    Le convertisseur applique les règles suivantes (identiques à ce que gère l'app ITM) :

    • Type "bouteille" (saisie au nb de pièces) : produits dont le nom Yokitup commence par B. M. J. MATHU. NABU. 1/2. BABY. MIDAS. BALTHA. REHO. BIB. COFFRET. GALLON.
    • Type "litre" (saisie aux 4 formats 50/70/75/100cL) : produits sans préfixe et avec unité cL/L/ml dans Yokitup
    • Type "unit" (saisie au nb de pièces) : produits sans préfixe et avec unité unit, kg, g
    • Les produits sans tag établissement sont conservés mais marqués comme archives (non visibles dans les inventaires actifs)
    • Overrides : CORONA 33 CL VP 4,5° forcé en BTL, GINGER BEER VERSE forcé en LITRE
